## 🤖 機械学習ってなに？

機械学習とは、**大量のデータからコンピュータが自動的にルールを学習する技術**です。

たとえば、人間が「犬」と「猫」を見分けるとき、耳の形・鼻の形・体型などの特徴を無意識に使っています。機械学習では、これと同じことをコンピュータにやらせます。
大量の画像（データ） → モデルが学習 → 新しい画像を見て予測

### 🧠 ニューラルネットワークとは？

機械学習の中でも、人間の脳の神経回路を模倣した手法が**ニューラルネットワーク**です。
入力層          中間層（隠れ層）        出力層
[X線画像] → [特徴を抽出] → [特徴を変換] → [正常 or 肺炎]

画像を数値の集まりとして受け取り、層を重ねるごとに「エッジ」→「形」→「パターン」と抽象的な特徴を学習していきます。

### 📊 学習の仕組み

1. **予測する** : モデルが画像を見て「正常」か「肺炎」かを予測
2. **誤差を計算する** : 正解と予測のズレ（損失）を計算
3. **修正する** : ズレが小さくなるようにモデルの内部パラメータを更新
4. **繰り返す** : これを何度も繰り返すことで精度が上がる

> この更新の仕組みを**バックプロパゲーション（誤差逆伝播法）** と呼びます。

### 🖼️ 画像認識に使うCNN

このノートブックでは **CNN（畳み込みニューラルネットワーク）** を使っています。CNNは画像の特徴を効率よく抽出するのが得意で、医療画像・顔認識・自動運転など幅広い分野で使われています。

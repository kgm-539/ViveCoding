# 🫁 Chest X-Ray Pneumonia Classifier

肺のX線画像から**正常 / 肺炎**を分類する画像分類モデルを、Google Colabで学べるノートブックです。
---

## 📌 このノートブックでできること

- 胸部X線画像データセットの読み込みと可視化
- PyTorchを使った画像分類モデルの構築・学習
- 学習済みモデルによる予測結果の確認

## 🎯 対象者

- 機械学習をはじめて学ぶ方
- Google Colabをはじめて使う方
- Pythonの基本的な読み書きができる方

## 🚀 使い方

1. 上の **Open in Colab** ボタンをクリック
2. Googleアカウントでログイン
3. メニューから **「ランタイム」→「すべてのセルを実行」** を選択
4. あとは自動で実行されます！

> **💡 Google Colabとは？**  
> ブラウザ上でPythonを実行できる無料サービスです。インストール不要で、GPUも無料で使えます。

## 🛠️ 使用技術

- Python 3
- PyTorch
- torchvision
- matplotlib
- Google Colab

## 📂 データセット

[Chest X-Ray Images (Pneumonia) - Kaggle](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)
